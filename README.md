## About
An Angular project to display a page to view contacts and add addresses.


## Installing and running for the first time locally

1. On root of the project run: 
```
npm i
```
2. Start the server: 
```
ng serve
```
3.  Navigate to 
```
http://localhost:4200/
```
4. Run json server seperately
```
json-server contacts.js
```
